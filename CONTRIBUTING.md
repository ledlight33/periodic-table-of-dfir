# Contributing to THE PERIODIC TABLE OF DFIR

Thanks for your interest in improving this project! Here's how you can help.

## Ways to Contribute

### 1. Suggest a New Tool
Open an issue with:
- Tool name and official URL
- Which category it belongs to
- License type (os/cm/fr/fm/std)
- A one-line description
- Why it should be included

### 2. Report an Error
Open an issue if you notice:
- Wrong URL for a tool
- Incorrect license type
- Outdated tool (abandoned/archived)
- Typo or broken link

### 3. Propose a Change
For bigger changes (new categories, layout changes, feature ideas), open an issue first to discuss before submitting code.

### 4. Submit a Pull Request
1. Fork the repository
2. Create a feature branch (`git checkout -b fix/update-tool-xyz`)
3. Make your changes
4. Test locally to make sure nothing breaks
5. Submit a PR with a clear description of what and why

## Guidelines

- Keep the total at 118 tools (matching the real periodic table). To add a tool, propose which one it replaces and why.
- Every tool must have: number, symbol (2 chars, CamelCase), full name, category, license, URL, description.
- Symbols must be unique (case-insensitive).
- Don't submit commercial tools without verifying they're still actively sold/maintained.

## Code of Conduct

Be respectful, constructive, and professional. We're all here to make DFIR better.
